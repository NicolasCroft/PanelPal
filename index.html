Moon Robot Project

Welcome to NOthing This mobile robot is designed to do nothing. It features custom 3D-printed feet and a brush assembly, along with special gecko-inspired adhesive for reliable traction in low-gravity, dusty environments.

3D-Printed Parts

I designed and printed two main parts:

  • Robot Feet with gecko adhesive technology.
  • Brush Assembly for sweeping dust off the solar panels.

Gecko Adhesive Technology

The feet of the robot use a specialized gecko adhesive concept, which emulates the setae on a gecko’s feet. This adhesive is highly effective for clinging to surfaces even in low-pressure or microgravity environments. By implementing this on the robot’s feet, the robot can maintain stable contact with lunar solar panels or other surfaces without slipping.

Python Code

All motion and sensor logic is controlled by a Python script running on a Raspberry Pi. Below is a snippet from the main control file: 


# robot_control.py

import time
import RPi.GPIO as GPIO

# Example pins for motors
LEFT_MOTOR_PIN = 17
RIGHT_MOTOR_PIN = 18

GPIO.setmode(GPIO.BCM)
GPIO.setup(LEFT_MOTOR_PIN, GPIO.OUT)
GPIO.setup(RIGHT_MOTOR_PIN, GPIO.OUT)

def forward(duration):
    GPIO.output(LEFT_MOTOR_PIN, GPIO.HIGH)
    GPIO.output(RIGHT_MOTOR_PIN, GPIO.HIGH)
    time.sleep(duration)
    GPIO.output(LEFT_MOTOR_PIN, GPIO.LOW)
    GPIO.output(RIGHT_MOTOR_PIN, GPIO.LOW)

def main():
    print("Starting cleaning sequence...")
    # Move forward for 2 seconds
    forward(2)
    # Additional logic for cleaning routine
    # ...
    print("Cleaning complete!")

if __name__ == "__main__":
    main()
    GPIO.cleanup()
